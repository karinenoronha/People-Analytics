Project Objective

To identify the key factors influencing employee attrition and support strategic HR decision-making.

Tools

Python
Google Colab
Pandas
Seaborn
Matplotlib
Power BI

KPIs

Turnover Rate
Headcount
Average Salary
Average Tenure
Average Age
Key Insights
Employees working overtime showed a higher attrition rate.
The Sales department had the highest turnover rate.
Low job satisfaction was strongly associated with employee attrition.
